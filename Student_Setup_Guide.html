STUDENT SETUP & DEPLOYMENT GUIDE

SecureStego Messenger

Everything a student needs to install and run this project on their laptop

1
Install These First
  • 01
    Python 3.11 — download from python.org/downloads
    ⚠ During install — tick the box "Add Python to PATH"
  • 02
    VS Code — download from code.visualstudio.com
  • 03
    Python Extension inside VS Code — open VS Code → Extensions tab → search Python → install
2
How to Get the Project
💾
Via USB
Copy securestego_messenger folder directly to D:\ drive
📲
WhatsApp / Google Drive
Send as ZIP → Download → Right click → Extract All → to D:\
🌐
Online (No Setup)
Deploy to pythonanywhere.com — student just opens a link in browser
3
Run the Project — Commands
  • 01
    Open PowerShell — press Windows Key + R → type powershell → Enter
  • 02
    Go to project folder
    PS> cd "D:\securestego_messenger"
  • 03
    Install all required packages
    PS> pip install django python-dotenv cryptography Pillow whitenoise gunicorn
  • 04
    Create the database — run once only on first setup
    PS> python manage.py migrate
  • 05
    Start the server
    PS> python manage.py runserver
  • 06
    Open browser and go to
    http://127.0.0.1:8000
4
Important Rules
✗Never delete the .env file — project will crash without it
✗Never run urls.py, views.py, settings.py, models.py directly with python
✗Never close PowerShell while server is running
✓Only ever run manage.py — that is the only file you run
✓Terminal must always be inside the securestego_messenger folder
✓Run migrate only once on the very first setup
SAVE THIS FILE AT — D:\securestego_messenger\static\images\

Project runs at http://127.0.0.1:8000 — always start server with python manage.py runserver
