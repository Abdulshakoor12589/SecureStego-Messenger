SUPERVISOR PRESENTATION GUIDE

SecureStego Messenger

Complete step-by-step guide for showing your project to supervisor or client

1
Day Before — Preparation
  • 01
    Open PowerShell and start your server to confirm everything works
    PS> cd "D:\securestego_messenger"
    PS> python manage.py runserver
  • 02
    Open browser and check all pages load correctly
    http://127.0.0.1:8000
  • 03
    Create a demo account — go to signup page and register with a username and password you will remember
    http://127.0.0.1:8000/signup/
  • 04
    Take screenshots of all pages and save them in D:\securestego_messenger\static\images\ as backup
  • 05
    Charge your laptop fully and practice the demo at least 2 times
2
On The Day — Before Supervisor Arrives
  • 01
    Press Windows Key + R → type powershell → press Enter
  • 02
    Go to project folder
    PS> cd "D:\securestego_messenger"
  • 03
    Start the server
    PS> python manage.py runserver
    Wait until you see: Starting development server at http://127.0.0.1:8000/
  • 04
    Open Chrome or Edge browser and go to
    http://127.0.0.1:8000
  • 05
    NEVER close PowerShell while showing the demo — server stops if you close it
3
In Front of Supervisor — Demo Order
Step Page URL What to Say & Do
01 Home Page /
"This is SecureStego Messenger — a secure system combining Encryption and Steganography. Normal encryption hides message content. Our system also hides the existence of the message inside an image."
02 Signup /signup/ Register a new account live.
"Passwords are stored as hashed values — never plain text. Django handles secure authentication."
03 Dashboard /dashboard/
"The dashboard shows all operations, encryption count, and security status."
04 Encrypt /encrypt/ Type: "This is my secret message for the demo"
Click Encrypt. Copy and save the key shown.
"Message encrypted using Fernet AES-128. The key is never stored on the server — sender must share it privately."
05 Embed /embed/ Upload any photo from laptop. Download the stego image.
"Encrypted message is hidden in the image using LSB Steganography — changing only the last bit of each pixel. Zero visible difference to the image."
06 Extract /extract/ Upload the stego image you just downloaded.
"The system reads pixel data and extracts the hidden encrypted payload."
07 Decrypt /decrypt/ Paste payload + enter the saved key. Click Decrypt.
"Original message recovered. Full secure communication cycle is complete."
08 Settings /settings/
"Users generate AES-256 keys here. Key history is tracked. Two-factor auth and session timeout available."
09 Security Log /security-log/
"Every action is logged — logins, encryptions, failed attempts. Full security audit trail."
4
If Something Goes Wrong — Quick Fixes
Problem Fix
Server stopped or PowerShell closed cd "D:\securestego_messenger" → python manage.py runserver
Page shows error or red screen Show screenshots from static\images\ folder as backup
Forgot login password Go to /signup/ and register a new account
Browser closed accidentally Reopen browser → type http://127.0.0.1:8000 (server keeps running)
Laptop goes to sleep Wake it up → reopen browser → go to http://127.0.0.1:8000
